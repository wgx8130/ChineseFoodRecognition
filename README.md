# Enhancing Dietary Management through Advanced Chinese Food Image Recognition: Performance Analysis and Optimization Strategies

We provide the source code of the paper "Enhancing Dietary Management through Advanced Chinese Food Image Recognition: Performance Analysis and Optimization Strategies" for readers. This paper has been submitted to the journal "The Visual Computer" for review.

## Training

- ResNet, DenseNet, ResNeXt, SENet, and Vision Transformer  

  You can directly change the dataset path and the corresponding model to train.
  
- ConvNeXt and ConvNeXt V2  

  Please install the corresponding packages first: 
  - ConvNeXt: https://github.com/facebookresearch/ConvNeXt
  - ConvNeXt V2: https://github.com/facebookresearch/ConvNeXt-V2   
  
  Then, you can fine-tune the models.
